## 🖥️ BBARIT Terminal

Run **Claude · Codex · Gemini · Kimi · Qwen · OpenCode · Ollama** side by side in real PTY terminals, and pair two of them into an **autonomous self-reviewing dev team**.

- 🦌 **Broker Agent** — one click opens a Developer + Reviewer AI pair, mediated by a transparent **mechanical broker** (no extra LLM cost). They coordinate through **git diffs + a Karpathy Wiki**, the developer signals `DONE`, the reviewer drives the rework loop, and on approval the broker **auto-merges an isolated worktree** — all without ever merging/deploying without your approval.
- 🧩 **Multi-harness manager** — run several dev/review pairs in parallel across projects, each worktree-isolated, with its own live status window; **hot-swap models** mid-session and **auto-resume** sessions on restart.
- 🧠 Multi-agent terminals, AI Roundtable, Orchestra / LangGraph workflow pipelines
- 🛠️ Git panel, hybrid code search + code-graph, Kanban/Gantt, MCP, MySQL/Postgres, PM2, SSH
- 📚 Karpathy Wiki (project + global), notes & knowledge graph, skills library, self-improving rules, RTM requirements tracking
- 🔌 Telegram · Discord · Slack · GitHub · Linear, **Web Terminal** (browser access via Cloudflare) and **real-time team Collab**
- 🎨 **NanoBanana AI studio** (icons, banners, logos, ads, video, web builder), native video editor, screen capture

Native, fast, auto-updating — built with **Tauri + Rust + React**.
