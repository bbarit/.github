## 🖥️ BBARIT Terminal

Run **Claude · Codex · Gemini · Kimi · Qwen · OpenCode · Ollama** side by side in real PTY terminals, and pair two of them into a self-reviewing dev team.

- 🦌 **Broker Agent** — one click opens a Developer + Reviewer AI pair that coordinate through **git diffs + a Karpathy Wiki**, review continuously, **improve themselves** over time, and never merge/deploy without your approval.
- 🧠 Multi-agent terminals, AI Roundtable, orchestra/workflow pipelines
- 🛠️ Git panel, hybrid code search, Kanban/Gantt, MCP, MySQL/Postgres, PM2, SSH
- 📚 Karpathy Wiki (project + global), notes & knowledge graph, skills library, self-improving harness
- 🔌 Telegram · Discord · Slack · GitHub · Linear, remote access
- 🎨 Native video editor, screen capture, media generation

Native, fast, auto-updating — built with **Tauri + Rust + React**.
