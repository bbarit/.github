## 왜 빠릿인가요

혼자서도 **빠릿하게 만들고**, **빠릿하게 런칭**할 수 있게.
강의 + 개발도구 + 마케팅 도구 + 커뮤니티를 한 번에 묶었습니다.

- **1인 1멤버십** · 1년 66만원 · 부가세 포함
- **카카오톡 결제(결제선생)** · 자동갱신 없음
- **단일 플랜** · 회원이 되면 9개 제품 전부 사용

## 시작하기

1. [bbarit.com](https://bbarit.com) 접속 → **GitHub로 시작하기** 클릭
2. 멤버십 결제 (카카오톡 결제 · 1년 66만원)
3. [포털](https://bbarit.com/app) 에서 빠릿터미널 · 빠릿캡처스 다운로드
4. [빠릿클래스](https://bbarit.com/app/class) 에서 강의 시작
