## BBARIT Terminal

The all-in-one AI vibe-coding IDE. Launch **Claude · Codex · OpenCode · Kimi · Gemini · Ollama**
inside a real native terminal, with file editors, code search, notes, Git, task boards,
MCP, and database tools — all in one desktop app.

- **Free** — sign up and download instantly. No payment required.
- **macOS (Apple Silicon)** and **Windows**. Intel Macs are not supported.
- 6 AI coding CLIs in one click · real PTY terminal (xterm.js + Rust portable-pty)
- 13 dedicated file editors · BM25 code search · notes & knowledge base
- Full Git panel · Kanban · Todo · Gantt · MCP servers · MySQL / Postgres
